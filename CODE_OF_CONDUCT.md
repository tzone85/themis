# Code of Conduct

The Themis project adopts the **Contributor Covenant, version 2.1**, as
its code of conduct.

## The canonical text

The full, authoritative version lives at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Source: <https://github.com/EthicalSource/contributor_covenant>

That document — not this file — is the binding text. Read it before
contributing.

## Scope

The Covenant applies to:

- Activity in this repository (issues, pull requests, discussions,
  reviews, commit messages).
- Public conversation about Themis where a participant is identifiable
  as a Themis contributor or maintainer.
- Private channels (email, DMs) used to coordinate on Themis work.

## Reporting

Report Code of Conduct violations to:

- **Email:** `thando.mini@sanlam.co.za`
- **Subject line:** `[themis conduct]`

Themis is currently maintained by one person. Reports are read by the
maintainer only. If the report concerns the maintainer themselves,
escalate through the Contributor Covenant's recommended channels at
the link above.

You can expect:

- Acknowledgement within 3 working days.
- A written outcome within 14 working days where the facts permit.
- Reporter identity kept confidential to the extent compatible with
  acting on the report.

## Enforcement

Enforcement follows the four-step "Enforcement Guidelines" section of
the Covenant verbatim (Correction → Warning → Temporary Ban →
Permanent Ban). See the canonical document for full text.

## Why adopted by reference

We adopt the Covenant by reference rather than copy its text into this
repository so that:

- Operators reading this file are sent to the authoritative version.
- Covenant updates do not require a documentation drift commit here.
- The reporting and scope sections — the parts unique to *this*
  project — are not buried inside the boilerplate.
